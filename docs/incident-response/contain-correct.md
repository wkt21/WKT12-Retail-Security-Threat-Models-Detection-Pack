# Step 4 — Contain and Correct

## Purpose
Once fraud or injection attempts are confirmed, contain the affected systems and correct vulnerabilities.

## Containment Steps
- Disable affected scanner or reader.
- Lock down POS terminal.
- Remove counterfeit labels or coupons.
- Freeze suspicious coupon codes.
- Isolate affected network segment.

## Correction Steps
- Update scanner configuration.
- Patch coupon parser logic.
- Deploy RFID/ESL honeytokens.
- Add SKU/discount binding rules.
- Update RF monitoring rules.
